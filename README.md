# Credit Risk Model

This project predicts whether a customer is a good or bad credit risk using machine learning.

## Key Features
- EDA & preprocessing
- SMOTE for imbalance
- Model comparison
- Logistic Regression (final model)
- Threshold optimization

## Results
- Recall: 85%
- F1 Score: ~0.57

## Tools
- Python, Scikit-learn, Pandas, NumPy
